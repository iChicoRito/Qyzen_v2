# Objective: Revise the assessment modal layout

---

## Description
Modify the assessment creation modal to remove the tabbed interface and return to a single-modal layout. Currently, the modal organizes fields into separate tabs based on category or purpose. The revised design should display all fields within a single modal view while maintaining consistent layout and preserving all existing functionality.

---

## Primary Objective
Remove the tabbed structure from the assessment creation modal and consolidate all fields into a single modal view.

---

## Secondary Objectives
- Ensure the layout remains consistent throughout the modal
- Maintain all existing functionality from the tabbed version

---

## Supporting Tasks

### Layout Restructuring
- Remove all tabs currently used to separate fields by category or purpose
- Display all assessment fields together within the main modal body
- Organize fields in a coherent, user-friendly layout

### Consistency and Functionality
- Maintain consistent styling and spacing across all fields in the new layout
- Preserve all functionality that existed in the tabbed version (form inputs, validation, submission, etc.)
- Ensure all fields remain accessible and usable in the single-view format

---

## Detailed Breakdown

### Current State
The assessment creation modal uses tabs to organize fields by category or purpose, with each tab containing related fields.

### Desired State
All assessment fields should be displayed within the modal without tabs, reverting to the previous layout where all content appears in a single view.

### Requirements
- The modal should present all necessary fields at once
- No tabs or tab-based navigation should remain
- All features (field inputs, validation, save/submit actions) must work exactly as they did with tabs
- Layout should be clean, consistent, and easy to follow

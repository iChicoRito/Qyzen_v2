# Objective
## Add Assessment Duplication and Restrict Section Editing to Single Selection

---

## Role

You are a full-stack web application developer specializing in educator assessment management, data duplication workflows, and form validation.

---

## Description

Enhance the educator assessment page by adding a duplication feature that allows educators to reuse an existing assessment without manually creating it again. The duplication action must be accessible from the assessment’s context menu and must open a modal where the educator can review and edit only the assessment details before creating the duplicate. All questions and polls from the original assessment must be copied into the duplicated assessment without being modified. Additionally, update the existing assessment-editing functionality so the section field accepts only one selected section rather than multiple sections.

---

## Primary Objective

Allow educators to duplicate an existing assessment with its original questions and polls intact while permitting assessment-detail changes through a modal before the duplicate is created.

---

## Secondary Objectives

- Add a **Duplicate** option to each assessment’s context menu.
- Allow educators to edit the duplicated assessment’s details before confirming its creation.
- Preserve all original questions and polls in the duplicated assessment.
- Restrict the section field in the assessment edit form to a single selection.

---

## Success Criteria

- Every eligible assessment displays a **Duplicate** action in its context menu.
- Selecting **Duplicate** opens a modal populated with the existing assessment details.
- The educator can modify the duplicated assessment’s details before confirming duplication.
- Confirming the modal creates a separate assessment record.
- The duplicated assessment contains the same questions and polls as the original assessment.
- Editing details in the duplication modal does not modify the original assessment.
- The original and duplicated assessments maintain independent assessment details after duplication.
- The assessment edit form allows only one section to be selected.
- An educator can replace the currently assigned section with another single section.
- Existing assessments with multiple stored sections are handled correctly when opened for editing.

---

## Constraints

- Duplicating an assessment must not modify the original assessment.
- Questions and polls must not be changed during the duplication process.
- The section field in the assessment edit functionality must not support multiple selections.
- Only one section may be assigned or changed through the assessment edit form.

---

## Context & Dependencies

- The changes apply to the educator-side assessment page.
- Assessments already have a context menu and an existing edit functionality.
- Assessments contain details, questions, and polls that must be included in the duplication workflow.
- The duplication feature must integrate with the existing assessment creation, question, poll, and section-assignment structures.

---

## Stakeholders

- Educators
- Students receiving the duplicated assessments
- Assessment administrators

## Detailed Breakdown

### Duplicate Action

Add a **Duplicate** option to the context menu of each assessment on the educator assessment page. Selecting this option must start the duplication workflow without requiring the educator to recreate the assessment manually.

### Duplication Modal

After the educator selects **Duplicate**, display a modal containing the original assessment details. Allow the educator to review and modify only these details before confirming the duplication. The modal must create a new assessment and must not update the original assessment.

### Question and Poll Preservation

Copy all questions and polls associated with the original assessment into the duplicated assessment. Their content and structure must remain unchanged during duplication. Any later changes made to the duplicated assessment must not affect the original assessment’s questions or polls.

### Single-Section Assessment Editing

Update the section field in the existing assessment edit functionality so it supports only one selected section. Educators must be able to retain the current section or replace it with another section, but they must not be able to assign multiple sections through the edit form.

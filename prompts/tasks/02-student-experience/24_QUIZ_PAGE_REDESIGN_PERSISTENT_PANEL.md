# Objective
## Quiz Page Redesign with Persistent Information Panel

---

## Description
The current quiz page is overly simple, lacking supplementary information and user interaction elements. The goal is to introduce additional content that enhances the user experience without disrupting or distracting the user while they are answering questions. The redesigned page will adopt the same two-panel layout used on the Result page. In that layout, the left panel displays a summary of the result, while the right panel contains attempt history and review options. For the quiz page, the left panel will be repurposed to present assessment and quiz details, including the timer, which will be relocated from its current position in the top navigation bar.

---

## Primary Objective
Redesign the quiz page to include a persistent, information-rich left panel using the existing two-panel layout from the Result page, ensuring users can access key details without interruption during the quiz.

---

## Secondary Objectives
- Relocate the timer from the top navigation bar into the new left panel.
- Make the design fully responsive for mobile devices.
- Provide a toggle button on mobile that allows users to hide or show the left panel as a sidebar.

---

## Success Criteria
- A two-panel layout is implemented on the quiz page, matching the structure used on the Result page.
- The timer is moved from the top navigation bar into the left panel.
- The left panel displays all specified details: timer, warning triggers, progress indicator, assessment details, and educator information.
- On mobile devices, a toggle button shows and hides the left panel as a sidebar.

---

## Constraints
- The addition of new content must not interrupt the user while they are answering quiz questions.

---

## Context & Dependencies
- The existing Result page already features the desired two-panel layout, which serves as the reference design for this update.

# Objective
## Implement Question Bank and Randomized Question Pool for Assessments

---

## Description
The current system lacks a centralized question bank, which is essential for storing and managing assessment questions. The goal is to implement a question bank where educators can create, store, and organize a collection of questions. A question pool feature will then leverage this bank to generate assessments by randomly selecting a subset of questions for each student. While all students take the same overall assessment—covering the same topics and difficulty—each receives a unique set of questions. This approach promotes assessment integrity, reduces answer sharing, and provides varied question exposure, all while allowing educators to reuse questions efficiently.

---

## Primary Objective
Implement a question bank and question pool that enables educators to store questions and deliver equivalent, randomized assessments to each student.

---

## Context & Dependencies
- The existing system currently lacks a question bank; this feature will be introduced as a new capability within that system.

---

## Stakeholders
- Educators: create, store, and manage questions in the question bank.
- Students: take assessments composed of randomized questions drawn from the question pool.

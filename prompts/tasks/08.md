# Objective
## Educator and Student Announcement Feature

---

## Role
You are a full-stack web developer responsible for implementing user interface components, following existing design patterns and component libraries.

---

## Description
Implement a complete announcement feature with distinct interfaces for educators and students, developed by two agents working concurrently and independently. The educator interface allows creating rich-text announcements with optional image uploads, targeting specific sections, subjects, or all enrolled students globally. The student interface displays these announcements in a dedicated sidebar page and within notifications, using specific card layouts sourced from an existing template. Both sides must integrate seamlessly with the existing navigation and component structure.

---

## Primary Objective
Build a bidirectional announcement system where educators create and target announcements with rich content, and students view them through a dedicated interface and notifications, with both the educator and student sides developed simultaneously by independent agents.

---

## Secondary Objectives
- Provide educators with a full-featured announcement creation tool including rich text editing and image uploads.
- Provide students with a dedicated announcement view and notification integration.
- Apply existing UI card patterns from the template for consistent student-side display.
- Enable parallel, independent execution of educator-side and student-side development tasks.

---

## Success Criteria
- A functional "Announcement" sidebar menu item appears on the educator side.
- Educators can create announcements via a modal with Title, optional Description, multiple image uploads (10MB max per file), rich text body, and a target section/subject selector.
- A global announcement switch uses the admin settings page switch card layout to show the announcement to all students enrolled with that educator.
- The educator announcement page displays a table of all announcements with a "Create Announcement" button matching existing table page patterns.
- A functional "Announcement" sidebar menu item appears on the student side.
- Student announcements appear in the notifications area.
- Announcements with images render using the card pattern from `demo1/public-profile/profiles/feeds.html` matching the Jenny Klabber card with description "Now that I’m done thoroughly mangling that vague metaphor, le...".
- Announcements without images render using the Jenny Klabber card with description "This doesn’t mean that all bloggers are insince...".
- All student-side announcement cards omit the three-dot context menu icon and omit any like or comment functionality.

---

## Constraints
- Image uploads are limited to a maximum of 10MB per file.
- Announcements target only students enrolled with the creating educator.
- Student cards must display only; no comments, likes, or context menu icons.

---

## Context & Dependencies
- The educator creation modal must reuse existing modal components.
- The global announcement switch layout must reuse the layout from Admin Side > Settings Page switch cards.
- Student announcement cards must be derived from `demo1/public-profile/profiles/feeds.html`.
- Both agents will work at the same time independently; educator-side and student-side tasks have no sequential dependency on each other.

---

## Supporting Tasks

### Educator Side (Agent 1)
- [Tag: Parallel] Add "Announcement" menu item to the educator sidebar.
- [Tag: Parallel] Build the announcement list page with a table displaying all announcements and a "Create Announcement" button matching existing table page styles.
- [Tag: Parallel] Implement the creation modal with fields: Title, Description (Optional), multiple image upload, Announcement Body/Content (rich text editor/WYSIWYG).
- [Tag: Parallel] Implement a section/subject selector to target the announcement.
- [Tag: Parallel] Implement a global announcement switch using the admin settings switch card layout.

### Student Side (Agent 2)
- [Tag: Parallel] Add "Announcement" menu item to the student sidebar.
- [Tag: Parallel] Display announcements in the student notification area.
- [Tag: Parallel] Display announcements with images using the specified card from `demo1/public-profile/profiles/feeds.html`.
- [Tag: Parallel] Display text-only announcements using the alternate specified card from the same template.
- [Tag: Parallel] Remove the three-dot context menu icon from all announcement cards.
- [Tag: Parallel] Ensure no comment or like interactions are present on any announcement card.
